# Nano Shopping List
Useful sopping list with mail sender.

# Tech
Written in Python2 using Tkinter, Pickle and smtplib

# Setup

Fedora 25 packages needed
- tkinter (for python2)
- sendmail

Enable sendmail server

`systemctl enable sendmail.service`


# Usage
`python nsl.py`
